# square-maker
making squares with '#'

Creates squares of '#' based on given numbers. If multiple arguments are given, it creates squares starting at the same point

## Examples

Argument: 5

````````````````
#####
#   #
#   #
#   #
#####
````````````````

Arguments: [7 3 9]

````````````````
#########
# #   # #
###   # #
#     # #
#     # #
#     # #
####### #
#       #
#########

````````````````
